# Projeto-Cronos---Ferramenta-de-Analise-T-cnica-
O projeto Cronos tem por objetivo desenvolver uma ferramenta de análise financeira totalmente gratuita, que possa ser executada em qualquer dispositivo com acesso à internet. A aplicação converterá dados históricos de criptoativos em uma interface gráfica interativa voltada à análise técnica.
